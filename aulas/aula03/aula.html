Aula 03 - Estruturas Condicionais

1. Introdução às Estruturas Condicionais

As estruturas condicionais permitem que um programa execute diferentes ações com base em condições específicas.

2. if...else

A estrutura `if...else` permite que você execute um bloco de código se a condição for verdadeira e outro bloco se a condição for falsa.

```javascript var idade = 18; if (idade >= 18) { console.log("Você é maior de idade."); } else { console.log("Você é menor de idade."); } ```

3. Operador Ternário

O operador ternário é uma forma simplificada de escrever estruturas `if...else`.

```javascript var idade = 18; var mensagem = (idade >= 18) ? "Maior de idade" : "Menor de idade"; console.log(mensagem); ```

4. switch

A estrutura `switch` é utilizada quando se tem múltiplas condições a serem avaliadas.

```javascript var diaSemana = 3; switch (diaSemana) { case 1: console.log("Domingo"); break; case 2: console.log("Segunda-feira"); break; // ... outros casos default: console.log("Dia não encontrado"); } ```

5. Exercício Prático

Desenvolva um código que, com base na temperatura, exiba mensagens diferentes, como "Está quente", "Está frio" ou "Temperatura agradável". Utilize estruturas condicionais.
